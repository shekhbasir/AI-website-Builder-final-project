# 🤝 Contributing Guidelines

Thank you for your interest in contributing to this project! 🚀
We welcome contributions from everyone.

---

## 📌 How to Contribute

### 1. Fork the Repository

Click on the **Fork** button at the top right of this repository.

---

### 2. Clone Your Fork

```bash
git clone https://github.com/shekhbasir/AI-website-Builder-final-project.git
cd AI-website-Builder-final-project

```

---

### 3. Create a New Branch

```bash
git checkout -b feature/main
```

---

### 4. Make Changes

* Add your feature or fix a bug
* Ensure your code follows clean coding practices
* Test your changes before committing

---

### 5. Commit Your Changes

```bash
git add .
git commit -m "Added: your feature name"
```

---

### 6. Push to Your Fork

```bash
git push origin feature/your-feature-name
```

---

### 7. Create a Pull Request

* Go to your forked repository
* Click on **Compare & Pull Request**
* Add a proper description of your changes

---

## 🧠 Contribution Ideas

* Improve UI/UX
* Add new AI features
* Optimize backend performance
* Fix bugs
* Improve documentation

---

## 📏 Coding Guidelines

* Use meaningful variable names
* Follow consistent formatting
* Keep functions small and modular
* Write reusable code

---

## 🔐 Security Guidelines

* Never commit `.env` files
* Do not expose API keys
* Use environment variables for secrets

---

## 🐛 Reporting Issues

If you find a bug:

1. Go to the **Issues** tab
2. Click **New Issue**
3. Describe the problem clearly

---

## 💡 Suggestions

Feel free to suggest new features or improvements by opening an issue.

---

## 🙌 Thank You

Your contributions make this project better! ⭐
